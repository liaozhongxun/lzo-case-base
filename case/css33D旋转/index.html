3D图形的简单制做

1、perspective:600px; 3d环境的父级 可简单理解为视距

2、构造3d环境(旋转这个3d环境的盒子带动你做好的3d图形)
transform-style:preserve-3d(它的默认值是flat 也仅有这两个属性)

3、给3d环境元素下的子元素rotate translate scale skew等效果调成自己想要的3d形状

4、动画
(1)先定个规则 @-webkit-keyframes 规则名{}
   规则的写法一 @-webkit-keyframes 规则名{
                   from{}to{} 从哪到哪
                }
   规则的写法二@-webkit-keyframes 规则名{
                   10%{怎样怎样}
                   20%{怎样怎样}
                   n%{怎样怎样}。。100%{怎样怎样} 
                }
(2)使用 定义一个类 
.类名{
	 -webkit-animation:规则名 多少时间完成 速度曲线 延迟 播放次数等;
 }
 (3)把这个类给要这样动的元素就行了
 //backface-visibility:hidden; 该属性定义当元素不面向屏幕时是否可见。
